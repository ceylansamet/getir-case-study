# Project Setup

Clone getirApp project.
```sh
$ git clone git@github.com:ceylansamet/getir-case-study.git
```
Install depencies.
```sh
$ npm install
```
### Dependencies

  - Express
  - dotenv
  - chai
  - async
  - express-validator
  - mongoose
  - express-swagger-generator

### Unit Test


 ```sh
$ npm test
```

 ### Run Project
 ```sh
$ npm start or node bin/www
```
